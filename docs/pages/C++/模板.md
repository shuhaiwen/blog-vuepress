## 类模板
### 类模板类型
- 非类型参数模板
- 默认模板参数
- 模板类成员模板
- 模板作为参数

```c++
template<typename T>
class A1 {
public:
    using _TYPE = T;
    _TYPE m_a;
    //A1():m_a(){}
    A1() = default;
    A1(const T& a):m_a(a){}
    A1& operator =(A1& a)
    {
        m_a=a.m_a;
        return *this;
    }
};
//模板作为参数
```
