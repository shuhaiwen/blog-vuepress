# bat批处理指令
- dir 　　　列文件名 
- cd　　　　改变当前目录 
- ren 　　　改变文件名 
- copy　　　拷贝文件 
- del 　　　删除文件 
- md　　　　建立子目录 
- rd　　　　删除目录 
- deltree　 删除目录树 
- format　　格式化磁盘 
- edit　　　文本编辑 
- type　　　显示文件内容 
- mem 　　　查看内存状况 
----------
*以下是新增加的命令* 
- help　　　显示帮助提示 
- cls 　　　清屏 
- move　　　移动文件，改目录名 
- more　　　分屏显示
