# 函数签名
*假设文件是Hello.java*
- 进入`Hello.java`文件所在目录
- 执行`javac Hello.java`命令，编译`Hello.java`,生成`Hello.class`文件
- 执行`javap -s Hello.class`
- 结果会打印在控制台
```sh
javap -s Hello.class

Compiled from "Hello.java"
public class Hello {
  public Hello();
    Signature: ()V

  public static void main(java.lang.String[]);
    Signature: ([Ljava/lang/String;)V
}
```
